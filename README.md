# Audio E-Book Using Python

This project converts a PDF file into an audio book using Python.  
The program lets you select any PDF from your computer and then reads all its pages using a text-to-speech engine.

## Features
- Select PDF using a file dialog
- Extract text from all pages
- Read the text aloud using pyttsx3
- Control voice speed, volume, and voice type
